官网做了一点小改动

发表于 2020-04-25

之前官网突然无法访问，在我们的修复下目前已经恢复正常了。

这次恢复正常后，官网做了一点改动，其实也没有啥大的改动，由于团队之前网页是外部成员外援制作的，加上平台化的原因，微博之类的平台早已成型，现在应该很少有人看官网了吧，官网纯粹就是情怀一下，标识着制作组的存在感。

官网首页最顶部的游戏换成了《勇者要生活》，是的这个游戏已经开始制作了，从4月1日开始启动项目的，目前进行了25天，这个项目是一个3D的游戏，也是Terobi制作组第一次尝试3D游戏的制作，这一次的3D不会很高大上，我们打算采用Q版，其实一开始打算做成真实比例的，后来预估了下，发现第一次做3D就提升这么多步子迈得太大了，我们连3D都还没入门了，所谓“连走路都要摔跤就想跑了那是不行了”。

好了，总之本期日志就更新到这里，官网日志不会经常更新，大家想看团队动态可以到微博，或者bilibili关注UP主枫2.

变装战机2的IOS版准备上线与后续计划

发表于 2019-09-20

最近经过一段时间的折腾，我们终于准备上线游戏的IOS版本了，这款小小的STG游戏花费了我们一年多的开发时间，而且这么长的时间我们才制作了100多个关卡，有时候我也在思考“我真是不会做游戏”，音乐一直重复我会觉得不爽，所以可能我们游戏的音乐量就比那些音游少点了，关卡一直不换场景也会不爽，所以我们一个场景居然只用了五关，场景只是更换贴图内容不变也不爽，所以我们结合场景出现于场景匹配的敌人，可能少有这种手游会做到每个关卡都有新要素，总之我们做到了。

游戏表现不是特别好，至少没有我想象中表现的好，其中主要的受挫是在googleplay平台上没有取得很好的成绩，因为我们缺乏买量的经济实力，但没有想到就是因为缺少这个下载量就会那么低，至今google上才1000多次下载。

我不知道IOS是不是也是必须要买量用户才能看见游戏，如果是的话IOS发布了也是凉凉吧，有时候游戏没人玩，真的不是你游戏不好玩，而是平台没用户看到你的游戏，你让用户怎么去玩呢，这不是酒好不怕巷子深的问题，而是巷子都不给你的问题。

关于后续，变装战机2是一款单机游戏，是的，这是一款拥有STAFF的游戏，至今为止离STAFF还有一个世界的内容，在STAFF后会进行二周目，二周目的设计并不是就是增加难度就好了，让我们拭目以待，Terobi的游戏可以说是从来没让玩家失望过，虽然我们的用户不多。

另外就是，新项目在策划阶段了，我们会做个3D付费PC游戏，因为我们做游戏一向会选择小众题材，所以我们决定前往付费市场，因为付费市场至少小众或许还能让我们有一些收入，在手游平台，小众要实现盈利太难了，广告费转化的收入太低了，所以接下来我们会非常忙，我们目前主力是3个人，我们3个人需要同时开发新游戏并继续更新变装战机2，但愿我们能忙过来吧。

写在变装战机2国内测试后

发表于 2019-04-17

因为日志一直没人关注，所以暂时就没有更新了，不过近期因为新项目《变装战机2》的发布，我们也就再更新一篇。

《变装战机2》是一款不起眼的小游戏，甚至我们在做的时候都一度怀疑这游戏还有人玩吗。

然而事实证明，游戏是有人玩的，甚至在国内小规模测试后，在短短3天时间内就超过2万玩家，游戏上线第五天，每日活跃玩家数据几乎没下降。当然以上并不是我们实力强，就是我们运气好，运气这玩意是玄学。

这对于一个独立小团队来说，是可观的数据了，但是我们深知游戏缺点和不完善太多，团队4个成员开发能力有限，不过还是在努力加内容加玩法，只是比起那些大团队，我们的步伐会慢一点。

接下来，游戏还会按照计划在海外上线，我们已经找到了地道的英语翻译为游戏打造英文版，面对海外更强烈的竞争，我们还在充分准备中。

第三大关BOSS制作开始

发表于 2018-12-05

目前我们进度已到第三关BOSS战的制作，想起了上一个版本第二关BOSS战我们一共花了20天时间进行制作。

这款游戏最精髓的以及最独特的地方也就是BOSS的设计，本次BOSS是一个水晶魔物，我们用2D帧动画描绘了BOSS的行为。（但是文章里并不打算发gif图向大家展示）

这次BOSS会包含5-6个攻击机制，同样的在3个难度中会拥有3种不同的机制，在简单难度中拥有5个行为，在困难难度以上会增加1种行为，以及噩梦难度下设计了BOSS狂暴时间。

于是，大约在2018年的最后一个月中，我们会花费大半个月的时间来制作这场BOSS战。

当然，除此之外，目前我们已经准备好第四大关的美术与BOSS设计了，同时主机的技能动画也制作好了一半，待0.3版发布后，游戏又会有一系列巨大的变化，只不过，我们计划月末先发布0.25版。

关于飞机配置功能

发表于 2018-11-17

之前还未向大家展示过游戏中的飞机配置功能，简单来说则是你可以在游戏中配置你拥有的战机，进入关卡后，获得相应颜色的道具即可变成相应配置的战机，这也是《变装战机2》中的特色系统。

接下来的0.2版本中提供了12架各具特色的战机，你可以通过“飞机工厂”将这些飞机制造出来，而制造飞机是一种随机的行为，是欧还是非就看你的RP咯。

技能功能制作中

发表于 2018-11-15

目前我们这款游戏系统还很不全面，有几个重要的功能还没有加上去，比如技能，技能的意思就类似于早期街机游戏中飞机的蓄力攻击或炸弹（因为是手机游戏，为了简化操作难度，我们将技能做在一起了）

因为我们的飞机种类特别多，而且还有不同的品级区分，所以游戏的技能是按照三种模式来设计的，第一种技能是纯粹的子弹加速一段时间，第二种技能是类似早期街机游戏的技能，即发射导弹、激光等等行为，第三种技能类似于早期街机的炸弹，不仅有超高的攻击力还可以挡住敌人的子弹，当然除此之外，我们会为部分飞机设定特别的技能，比如保护遮罩等。

0.2体验版预计近几天就能发布，尽请期待吧。
